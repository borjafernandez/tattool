# Dataset list

- Create new dataset
- Delete dataset
- Update dataset

# Create new dataset

- Can create a new dataset
- Can add a new column while editing
- Can remove column while editing
- Can change column name while editing
- Cannot save with empty column name

# Delete dataset

- Can delete dataset

# Update dataset

- When deleting dataset column, if column is mapped into recording, ask first, yes => #1, no => #2
- #1 delete mappings first then delete column
- #2 do not delete column
- When updating dataset column name, if column is mapped into recording, ask first, yes => #1, no => #2
- #1 update mappings with new name
- #2 do not change column name
